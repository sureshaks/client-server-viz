Introduction

This is an interactive visualization to explore the performance of Logistic Regression on the Blood Transfusion Data.

Usage

The interface provides options to tune two parameters -
1. Preprocessing scaler : The type of scaling to be used on the data. Options are Standard Scaler or Min Max Scaler
2. Regularization (C) : Inverse of regularization strength. It must be a positive float. Smaller values specify stronger regularization.

ROC curve

A receiver operating characteristic curve, or ROC curve, is a graphical plot that illustrates the diagnostic ability of a binary classifier system as its discrimination threshold is varied.The ROC curve is created by plotting the true positive rate (TPR) against the false positive rate (FPR) at various threshold settings. The true-positive rate is also known as sensitivity, recall or probability of detection in machine learning. The false-positive rate is also known as the fall-out or probability of false alarm and can be calculated as (1 - specificity).
